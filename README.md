# GAN-Based-Augmentation-for-Medicinal-Plant-Recognition
Evaluating cGAN and cWGAN-GP augmentation for deep learning-based recognition of Philippine medicinal plant leaves using VGG19 and ResNet50.
This repository contains the notebooks used for the thesis:

**Evaluating GAN-Based Augmentation for Deep Learning-Based Recognition of Philippine Medicinal Plants**

The study evaluates cGAN and cWGAN-GP as data augmentation methods for Philippine medicinal plant leaf classification using VGG19 and ResNet50 transfer learning models.

## Dataset

This study uses the PhilMedic: Philippine Medicinal Plant Leaf Dataset containing 40 classes of Philippine medicinal plant leaves.
